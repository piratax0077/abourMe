Mis inicios

Epoca de universidad

¡Hola! Mi nombre es Francisco Rojo y soy Desarrollador web. Mis inicios en la informática parten desde el año 2007 cuando ingrese a la Universidad del Mar a estudiar Ingeniería civíl en computación e informática . En aquellos tiempos empezámos con lenguajes clásicos como Pascal y C, luego pasamos a Java. Tambien se aplicaban herramientas de ofimática como Excel, Word y Access.

En el año 2013 por razones personales comence a estudiar nuevamente Ingeniería en Informática pero en el Instituto Profesional Santo Tomas en La Serena. En ese tiempo comencé a conocer lo que era la programación web con Java, Java EE. Otro lenguaje conocido fue PHP.

La culminación de mis estudias fue en el año 2019 cuando presenté un proyecto que consistía en el desarrollo de un Software como Servicio (SaaS) . Para empezár el proyecto consistía en dos aplicaciones. Una que funciona como servidor y otra como cliente. El servidor fue desarrollado con Java Web y el cliente consistía en una aplicación móvil híbrida utilizando el framework Ionic. La función principal era que en la aplicación móvil se mostrará un catalogo de productos y servicios que podian ser reservados para que posteriormente el usuario pueda retirarlo en tienda. Todas estas transacciones eran registradas en el servidor y almacenadas en una base de datos Mysql.

Ambas aplicaciones estan en mi repositorio Github
